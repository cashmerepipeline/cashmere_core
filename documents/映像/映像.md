# 映像定义

## 定义

    在不同 角度、 环境条件 、身份下, 由映像规则生成，指示管理、集合、实体属性的访问能力, 并且能在客户端缓存
    包括英文中的 reflect 和 view 两个含义
    实现时使用view

    形式:    
        { 管理映像 |  表映像 | 条目映像 }
    
    包含 管理-表-条目 三个级别的规则

    某种规则的描述
    也可以直接指定
    也可以动态生成

    管理表是特殊的一个表，所以管理级的映像是特殊的映像

    以映像为基础进行其他管理操作

## 映像规则

    管理规则
      对管理的规则
      管理访问入口规则
      在管理规则中指定

    表规则
      对表的规则
      表访问入口规则
      在表规则中指定

    实体规则
      对实体的规则
      在表级过滤规则中指定

    过滤规则
      在入口规则的基础上，对子级成员的过滤的映像规则
      对子级的访问上做进一步深入细化控制

## 规则计算

    规则 或
    规则 与
