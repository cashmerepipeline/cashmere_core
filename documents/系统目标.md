# 系统构架目标

    1. 实时迭代生产管理系统
    2. 完整覆盖 概念-->概念验证-->概念设计-->产品原型-->产品迭代-->最终交付 全过程
    3. 通用型基础平台, 提供基础组件和构架, 整个的流程设计组织单位根据自身情况定制
    4. 多国母语支持，完全支持utf-8字符集定义管理

# 实现方式

    后端, rust+grpc
    管理流程设计: python
    前端: 所有能支持grpc接口调用的工具, 自己独立研发
